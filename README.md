# Mesto
###### 0.0.1

Instagram prototype built to learn how Webpack works. 

Functionality:
* add new cards
* delete existing 
* toggle likes
* edit profile info

## Technologies used

* HTML 
* CSS
* JS
* Webpack

## Installing

* Clone repo
* Run `npm init` to set the dependencies
* Use `npm run build` for Prod version
* Use `npm run dev` for Dev version
